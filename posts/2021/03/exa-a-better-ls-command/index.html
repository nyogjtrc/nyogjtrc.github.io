exa 是 ls 指令的另一個選擇，使用 rust 開發的

github: https://github.com/ogham/exa

exa 的特色#

  • 幫檔案跟一些屬性都上色，更容易閱覽
  • 可以用樹狀圖顯示，tree 的另一個選擇
  • 可以支援 git status 以及一些額外的檔案資訊
  • 執行效能跟 ls 差不多

安裝方式#

1

$ sudo apt install exa

1

$ brew install exa
