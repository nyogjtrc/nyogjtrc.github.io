把 PATH 變數列成清單

在 Linux 的 Shell 環境中，要查看 PATH 變數時，通常都是 echo $PATH

當 PATH variable 中包含太多資料夾時，跟本看不懂

改以清單的方式顯示 PATH 內容

$ echo $PATH | tr ":" "\n"
/usr/local/sbin
/usr/local/bin
/usr/sbin
/usr/bin
/sbin
/bin

echo "${PATH//:/$'\n'}"

echo -e "${PATH//:/\n}"

直接用 echo 的方式好像會因 echo 的版本而有差異

列成清單之後發現一堆重複的路徑…
