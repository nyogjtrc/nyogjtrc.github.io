Install Lastest Firefox on Debian

February 10, 2020 · 1 min · nyo
Table of Contents
  • 下載
  • 建立 firefox.desktop
    • Reference

Debian 的套件庫中只有 Firefox-ESR，跟 Firefox 官網的版本有點差距

想要使用最新的 Firefox 就是，直接從官網下載安裝！！！

下載#

1
2

$ tar xvf firefox-72.0.2.tar.bz2
$ sudo mv firefox /opt

建立 firefox.desktop#

1

$ sudo vi /usr/share/applications/firefox-stable.desktop

 1
 2
 3
 4
 5
 6
 7

 9
10

[Desktop Entry]
Name=Firefox Stable
Comment=Web Browser
Exec=/opt/firefox/firefox %u
Terminal=false
Type=Application
Icon=/opt/firefox/browser/chrome/icons/default/default128.png
Categories=Network;WebBrowser;
MimeType=text/html;text/xml;application/xhtml+xml;application/xml;application/vnd.mozilla.xul+xml;application/rss+xml;application/rdf+xml;image/gif;image/jpeg;image/png;x-scheme-handler/http;x-scheme-handler/https;
StartupNotify=true

然後享用 Firefox
