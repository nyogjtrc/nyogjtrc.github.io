Set Sudo on Debian

安裝好基本款的 debian 是沒有 sudo 指令可以使用的，需要自行安裝設定

安裝 sudo

切換到 root 安裝

# su -
# apt install sudo

設定 sudoer

# usermod -aG sudo nyo

設定好之後，登入並用 sudo 隨意打個指令試試

$ sudo ls -l /root

成功！
